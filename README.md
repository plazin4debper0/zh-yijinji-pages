# zh-yijinji-pages

本仓库用于归档和发布多个独立的 HTML 页面。这些页面不隶属于任何特定域名或网站，仅作为静态内容的集中存储与展示。

## 项目简介

`zh-yijinji-pages` 是一个轻量级的 HTML 页面归档仓库。主要用途包括：

- 保存不便于长期维护的独立页面
- 提供临时的静态页面托管（通过 GitHub Pages 或其他类似服务）
- 作为个人或团队的小型页面备份库

所有页面均为独立文件，互不依赖，便于直接访问或引用。

## 目录说明

```
/
├── index.html          # 仓库主页（可选）
├── page-a.html         # 示例页面 A
├── page-b.html         # 示例页面 B
└── archive/            # 归档子目录（可选）
    └── old-page.html   # 旧版本或历史页面
```

- 根目录下存放当前活跃或常用的页面。
- `archive/` 目录用于存放不再更新但需要保留的页面。
- 你可以根据实际需要自行调整目录结构。

## 页面归档说明

- 每个 HTML 页面应尽量自包含（内联样式、脚本），减少外部资源依赖。
- 页面之间没有内容关联，均为独立展示。
- 建议在页面内添加适当的元信息（如 `meta` 标签），标明页面用途或版本。
